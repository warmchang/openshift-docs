= Troubleshooting etcd

[role="_abstract"]
{microshift-short} runs etcd as a managed, separate process to store cluster state. To ensure optimal performance and resolve issues, as an administrator, you can observe system activity and enforce memory usage limits by using the {microshift-short} configuration file.
