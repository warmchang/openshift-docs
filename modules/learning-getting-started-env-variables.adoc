= Creating additional environment variables

[role="_abstract"]
To simplify your command-line execution and reduce repetitive typing, configure environment variables for your cluster deployments. Using these variables makes it faster and more efficient to run the command to create a {product-title} cluster.

.Procedure
* Run the following command to set up environment variables:
+
[source,terminal]
----
$ export CLUSTER_NAME=<cluster_name>
$ export REGION=<VPC_region>
----
+
[TIP]
====
Run `rosa whoami` to find the VPC region.
====
