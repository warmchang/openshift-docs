After installing {product-title}, you can configure machines, cluster services, nodes, networking, storage, users, and alert notifications to meet your operational requirements.

After installing {product-title}, a cluster administrator can configure and customize the following components:

* Machine
* Bare metal
* Cluster
* Node
* Network
* Storage
* Users
* Alerts and notifications
