= Introduction to OpenShift updates
include::_attributes/common-attributes.adoc[]
:context: understanding-openshift-updates

toc::[]

[role="_abstract"]
With {product-title} 4, you can update an {product-title} cluster with a single operation by using the web console or the OpenShift CLI (`oc`).

Platform administrators can view new update options either by going to *Administration* -> *Cluster Settings* in the web console or by looking at the output of the `oc adm upgrade` command.
