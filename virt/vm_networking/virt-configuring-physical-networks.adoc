= Configuring physical networks

:context: configuring-physical-networks

toc::[]

[role="_abstract"]
As an OpenShift administrator, you can create or configure a physical network in the {product-title} web console without using the node network configuration policy (NNCP) page. When you use the physical network page in the web console, the NNCP is generated automatically. If you need more flexibility or require complex settings, use the NNCP page.
